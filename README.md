<!-- ABOUT THE PROJECT -->
## About The Project

This extension allows users to confirm their profile and get the approved icon using an 
identity number.

### Built With
* [Flarum](https://flarum.org/)
* [Laravel](https://laravel.com)



<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

* PHP Soap Extension
* PHP Curl Extension

### Installation

```sh
composer require cenkkoroglu/tcno-dogrulama
```
